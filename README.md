# Wede Android SDK (Kotlin)

Offline-first operational resilience SDK for Android.

## Features
- Offline dispatch with local scoring
- Guaranteed delivery queue (persists until server confirms)
- Bidirectional sync with sequence numbers
- Permanent device ID (survives reinstalls)
- Identical algorithm to backend and JS/RN SDKs
